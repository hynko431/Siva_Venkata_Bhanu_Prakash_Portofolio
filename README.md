This repository contains a Vite + React + TypeScript portfolio website.

Quick start

1. Install dependencies:

```powershell
cd "C:\Users\hulkh\Downloads\Siva_portofolio\portfolio-website-main"
npm ci
```

2. Run development server:

```powershell
npm run dev
```

3. Build for production:

```powershell
npm run build
```

Build output

The production-ready files are output to the `dist/` folder by Vite.

Deployment

This repository includes a GitHub Actions workflow that builds the site and deploys the `dist/` folder to GitHub Pages on pushes to `main`.

Notes

- If you want to serve the site from a custom domain, add a `CNAME` file into the `public/` directory before building.
- Node 18 is recommended (see `package.json` engines).
